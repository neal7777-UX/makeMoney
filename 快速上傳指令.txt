====================================
  休閒風個人記帳本 - GitHub 快速上傳指令
====================================

📌 第一步：在 GitHub 上創建新倉庫
- 網址：https://github.com/new
- 倉庫名稱：finance-tracker
- 不要勾選任何選項（README、.gitignore、License）

====================================

📌 第二步：在終端機執行以下指令

1. 進入專案目錄：
------------------------------------
cd "D:\AI學習庫\記帳軟體"
------------------------------------

2. 初始化 Git：
------------------------------------
git init
------------------------------------

3. 添加所有檔案：
------------------------------------
git add .
------------------------------------

4. 第一次提交：
------------------------------------
git commit -m "🎉 初始提交：休閒風個人記帳本 v1.0.0"
------------------------------------

5. 連接遠端倉庫（替換成您的資訊）：
------------------------------------
git remote add origin https://github.com/YOUR-USERNAME/finance-tracker.git
------------------------------------

範例：
git remote add origin https://github.com/johndoe/finance-tracker.git

6. 推送到 GitHub：
------------------------------------
git branch -M main
git push -u origin main
------------------------------------

====================================

✅ 完成！查看您的 GitHub 倉庫頁面確認上傳成功

====================================

📝 後續更新指令（修改程式碼後）：

------------------------------------
git add .
git commit -m "✨ 更新內容描述"
git push
------------------------------------
